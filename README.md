# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **n8n** and **Groq AI** that automatically evaluates resumes based on Applicant Tracking System (ATS) best practices.

This project allows users to upload a PDF resume through a web form. The workflow extracts the resume text, sends it to an AI model for analysis, and generates a professional ATS report with actionable feedback to help job seekers improve their resumes.

---

# ✨ Features

- 📄 Upload resumes in PDF format
- 🔍 Automatically extract text from PDF files
- 🤖 AI-powered resume analysis using Groq LLM
- 📊 Generate an ATS compatibility score
- 💪 Identify resume strengths
- ⚠️ Highlight weaknesses and missing keywords
- 📚 Recommend technical and professional skills to learn
- 💼 Suggest suitable job titles
- 📝 Provide personalized resume improvement recommendations
